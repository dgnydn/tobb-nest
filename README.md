# Türkiye Odalar ve Borsalar Birliği (TOBB) API

Bu API, TOBB tarafından yayınlanan verileri kullanıcıya sunmak için geliştirilmiştir.

## Kullanım

### Kurulum

```bash
git clone https://github.com/dgnydn/tobb-api.git
cd tobb-api
npm install
```

### Çalıştırma

```bash
npm start
```

### Deploy

```bash
vercel --prod
```
